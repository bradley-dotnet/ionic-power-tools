Supercharging Ionic Development

Welcome to the documentation pages for Ionic Power Tools. Here you'll find demos, example code, and our API reference to the must-have library for all Ionic Angular apps.

Components

Loading Spinner Quickly connect any RxJS observable to the ion-spinner component Dynamic Dialog Create a custom dialog that can pass strongly-typed data back to the caller using RxJS Form Input Adds validation, required *, and error messages (with on touched support) for form controls

Getting Started

Just install the package with npm install ionic-power-tools and import the modules you want to use!

Changes in 2.0

The DynamicRouterModule has been removed in favor of the ngx-strong-router library.
